# Changelog

All notable changes to Cordoval OS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release of Cordoval OS
- Full desktop environment with window management
- Window resize functionality (edges and corners)
- Window maximize/restore with double-click titlebar
- Window minimize and close functionality
- Integrated KDS app launcher with multiple apps
- Start menu with app categories
- Desktop icons for quick access
- Taskbar with app indicators
- System tray with time display
- Auto-update system integrated
- GitHub Releases support for distribution
- Windows installer (NSIS)
- Portable executable option
- Beautiful glassmorphic UI design
- Smooth animations and transitions
- Native system app launching (Explorer, Calculator, etc.)

### Technical
- Built with Electron + React + TypeScript
- electron-updater for automatic updates
- electron-builder for packaging
- Vite for fast development
- Window state management
- Multi-window support with z-index handling
